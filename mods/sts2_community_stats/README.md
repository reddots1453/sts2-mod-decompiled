# Stats the Spire — STS2 社区数据共享 Mod

**版本**: 0.13.1  
**适用**: Slay the Spire 2 (Early Access v0.99.x+)  
**类型**: 纯信息展示，不影响游戏平衡 

---

## 这个 Mod 做什么？

Community Stats 收集统计本地数据和收集所有订阅玩家的游玩数据，汇总后在游戏界面中实时展示社区统计信息。帮你做出更明智的选择 ——  或者，看看你的选择有多离谱。

---

## 功能一览

### 0. STS1代mod Infomod 和 Intent Graph 功能迁移和集成


### Infomod

**药水机会追踪**

顶部栏上的指示器跟踪当前的药水掉落几率。。将鼠标悬停在其上可看到精英战掉落概率和将有机会在多次战斗后看到至少一种药水。

**问号房间显示**

你可以悬停于未走过的问号房间上，看到下一个问号房间会发生什么的全部概率列表。影响问号遭遇概率的遗物也被纳入计算。

**卡牌掉落几率**

顶部栏上的指示器可以查看在下一个战斗奖励中看到某种稀有度类别的卡牌的可能性有多大。

**商店价格**
你可以悬停于未走过的商店房间上，显示在下一家商店中可以预期的价格。这些值采用颜色区分，可以轻松判断是否可以用当前的金币购买某种物品。影响商店物品价格的遗物也被纳入价格计算。




### Intent Graph

将怪物意图显示为状态机。鼠标悬停在怪物上，可以看到起意图状态机。

**意图**
显示怪物下次的行动。

**意图组**
用蓝色方框框起来的一组意图，是怪物下次所有可能的行动。
左上角是概率。
右上角是这个意图最多能连续使用的次数。

**意图变化**
总是从“初始”意图开始。
下回合，沿黄色箭头去下个意图或意图组。



### 1. 选卡界面 — 社区选取率 & 胜率

在卡牌奖励界面，每张卡牌下方显示：

```
选取 72% | 胜率 54%
```

- **选取**: 社区中该卡出现在奖励中时被选取的比例
- **胜率**: 选取该卡的玩家最终通关的比例
- 颜色编码：绿色(高胜率) / 黄色(中等) / 红色(低胜率)

### 2. 事件选项 — 社区选择率 & 胜率

在事件选项按钮旁显示：

```
选择 64% | 胜率 48%
```

### 3. 地图节点 — 遭遇战危险度

```
死亡 3.2% | 平均伤害 18
```

### 4. 战斗贡献统计图

**每场战斗结束后自动弹出**（F8关闭），展示本场战斗中每张卡牌、遗物、药水的贡献：

**全局汇总** 额外包含 **治疗** 分类：

```

  篝火休息          ██████████  20  (40%)
  [事件] 深渊浴场    ███  6  (12%)
```

**特色**：
- 卡牌(蓝色)、遗物(金色)、药水(青色) 在同一图表中混合排序
- **间接伤害追踪**：中毒、易伤增伤等归因到施加该效果的卡牌
- **力量/修饰器归因**：力量(紫色段)、易伤等加成不再归因到攻击牌，而是归因到提供力量的来源（如燃烧、恶魔形态）
- **Power 间接效果**：狂怒格挡、火焰屏障反伤、势不可挡伤害、狱火伤害等正确归因
- **防御追踪**：FIFO 格挡归因 + 敏捷加成 + 虚弱减伤 + 缓冲/无实体减伤
- **抽牌 / 能量获取** 独立显示各来源贡献
- **药水贡献**：力量药水、抽牌药水、格挡药水等均正确归因到对应统计
- **治疗统计**（全局汇总）：火堆休息、遗物(燃烧之血)、事件治疗/最大生命值增加、跨层恢复等分别显示
- **子条形图**：变形/生成的卡牌（如巨石牌、小刀）以缩进子条显示在来源卡下方
- **升级贡献**：武装等升级卡牌后，升级带来的额外伤害/格挡以橙色段标识
- `x3` 显示卡牌打出次数，`[R]` 标识遗物，`[P]` 标识药水

### 5. 全局贡献汇总

按 **F8** 可随时打开贡献面板，包含两个标签页：

| 标签 | 内容 |
|------|------|
| **上场战斗** | 最近一场战斗的详细贡献 |
| **本局汇总** | 本次 Run 所有战斗的累计贡献 |

### 6. 模组设置面板

按 **F9** 打开设置面板（也可从 Mod 设置屏幕进入）：

| 设置项 | 说明 |
|--------|------|
| **上传游玩数据** | 默认开启。关闭后不会向服务器上传任何数据 |
| **自动匹配进阶** | 自动匹配你当前的进阶等级附近的数据 |
| **最低/最高进阶** | 手动设置进阶范围 (0-20) |
| **最低玩家胜率** | 筛选高胜率玩家的数据 |
| **游戏版本** | 当前版本 / 所有版本 |
| **语言** | 中文 / English |

### 7. 商店购买记录

记录你在商店中的购买行为（卡牌、遗物、药水），显示社区购买率。

### 8. 卡牌移除 & 升级追踪

- 火堆升级卡牌时显示 **升级率**
- 商店/事件移除卡牌时显示 **移除率**

### 9. 遗物胜率

鼠标悬停在遗物栏的遗物上时，显示社区选取率和胜率。

### 10. 个人生涯统计

进入 **百科大全 → 角色数据**，新增的 **个人生涯统计** 标签展示你本地所有 RunHistory 的聚合数据：

- **胜率汇总卡片**：数据范围 / 进阶 W-L / 进阶胜率 / 最高连胜，最低进阶可调（0-10）
- **胜率趋势**：最近 10 / 50 / 100 / 全部 局滚动胜率
- **死因排行**：Top 10 死因，按战斗 / 事件 / 中途放弃 / 先古之民 区分图标
- **卡组构筑** + **路线统计**：每 Act 平均卡牌获取 / 购买 / 删除 / 升级、小怪 / 精英 / ? 房间 / 商店 / 火堆数
- **先古遗物选取率**：按 elder 分组，每个池子的遗物选取率 / 选取次数 / 胜率 / 胜率浮动
- **Boss 战损**：所有 4 幕 boss 的平均 HP 损失 + 死亡率（含 0 遭遇占位行）
- 角色筛选：全部角色 / 单个角色

### 11. 历史记录本局统计弹窗

进入 **历史记录详情**，右上角会出现 **📊 本局统计** 按钮。点击弹出居中弹窗，显示这一局的：

- 角色 · 进阶 · 楼层 · 胜负
- 卡组构筑 / 路线统计（每 Act 表格）
- 先古遗物选取（含 elder 头像 + 中文名 + 遗物图标）
- Boss 战损（每场 boss 战 HP 损失）
- **查看贡献图表** 按钮：回看本局所有战斗的贡献统计

弹窗支持 ✕ / Esc / 点暗背景关闭。

### 12. 卡牌图书馆 / 遗物收藏个人统计

在 **百科大全 → 卡牌图书馆 / 遗物收藏** 中，每张卡牌 / 遗物的详情面板会显示你的个人样本数 + 选取率 + 胜率 + 升级率 + 删除率 + 购买率，与社区数据并列对比。

## 数据说明

### 什么时候上传数据？

**第一次加载mod会上传本地历史记录数据**
**仅在每次 Run 结束时上传一次**（无论胜利或失败）。可在设置中关闭上传。

### 上传了哪些数据？

| 数据 | 说明 |
|------|------|
| 角色 & 进阶等级 | 用于分类统计 |
| 胜败结果 & 到达层数 | 用于计算胜率 |
| 每次卡牌奖励的选择 | 出现了哪些卡、选了哪张 |
| 事件选项选择 | 选了哪个选项 |
| 商店购买记录 | 买了什么、花了多少金币 |
| 卡牌移除/升级记录 | 移除/升级了哪张卡 |
| 每场遭遇战数据 | 敌人ID、受到伤害、回合数、是否死亡 |
| 战斗贡献数据 | 每张卡/遗物/药水的伤害/防御/抽牌等贡献 |
| 最终卡组 & 遗物 | Run 结束时的卡组和遗物列表 |

### 不会上传的数据

- **不会** 上传 Steam ID 或任何个人身份信息
- **不会** 上传你的系统信息或其他游戏数据

### 离线模式

如果无法连接服务器：
- 上传数据自动保存到本地队列，下次联网时重传
- 内置测试数据作为兜底。

## 安装

1. 将 `sts2_community_stats` 文件夹放入游戏目录的 `mods/` 文件夹下
2. 启动游戏，Mod 自动加载
3. 开始一局新 Run，统计数据会自动加载并显示

## 常见问题

**Q: 这个 Mod 会影响游戏平衡吗？**  
A: 不会。Mod 标记为 `affects_gameplay: false`，仅显示信息。

**Q: 我可以关闭数据上传吗？**  
A: 可以。按 F9 打开设置面板，取消勾选"上传游玩数据"即可。

**Q: 数据显示 "无数据" 是怎么回事？**  
A: 该卡牌/事件的样本量还不够，或测试数据中未包含该ID。
