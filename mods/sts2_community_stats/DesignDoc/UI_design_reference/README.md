## 如何使用

1. **UI 实现时**：打开对应截图作为视觉参考，确保新 UI 的布局、配色、交互与截图风格一致
2. **PRD/Plan 文档中**：使用"参考图N"格式引用，如"参考图4: STS1 Jaw Worm 意图状态机"
3. **Code Review 时**：对照截图检查实现是否符合设计意图

## 设计优先级

- **图1、图2**：STS2 原生 UI，**必须严格匹配风格**（面板背景、图标大小、标签颜色）
- **图4、图5、图6、图7、图8**：STS1 Mod 参考，**模仿交互模式和布局结构**，颜色/面板样式使用 STS2 原生值（见 `UI_STYLE_GUIDE.md`）
- **图3**：当前问题截图，仅用于说明需要修复的问题
