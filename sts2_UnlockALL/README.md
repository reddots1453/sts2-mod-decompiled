# STS2 Dev Tools - Unlock ALL

**版本**: 0.1.0
**适用**: Slay the Spire 2 (Early Access)
**用途**: 开发/测试辅助工具，一键解锁游戏全部内容

---

## 功能

按 **F7** 打开菜单，可选择：

| 按钮 | 功能 |
|------|------|
| **Unlock All Characters** | 解锁 Silent / Regent / Necrobinder / Defect |
| **Unlock All Cards / Relics / Potions** | 解锁所有卡牌、遗物、药水 + 图鉴标记可见 |
| **Unlock All Ascensions (A10)** | 所有角色进阶等级设为 10（单人+多人） |
| **Unlock All Timelines** | 解锁全部 57 个时间线节点 |
| **Unlock Everything** | 以上全部 + 怪物/事件图鉴 |

所有操作立即写入存档，**重启游戏后生效**。

---

## 安装

将 `sts2_UnlockALL` 文件夹放入游戏 `mods/` 目录：

```
mods/sts2_UnlockALL/
├── manifest.json
├── sts2_dev_tools.dll
└── README.md
```

---

## 注意事项

- 此工具修改的是 **modded 模式** 的存档（与无 mod 存档分离）
- 解锁操作不可逆（除非手动编辑存档或在游戏内重置时间线）
- 仅用于开发测试，不影响游戏平衡
